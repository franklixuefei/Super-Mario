SUPER MARIO
===========

<h2>How to play</h2>
<ul>
<li>first, make sure you have the X Library installed (just Google it :-P)</li>
<li>simply type <pre>make run</pre> to get started!</li>
<li>tap space to get start</li>
<li>left/right arrow to navigate direction</li>
<li>tap "j" to jump</li>
</ul>
